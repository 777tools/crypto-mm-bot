📍 APIキーの取り方

1. bitbank.cc にログイン
2. 右上メニュー →「API」を開く
3.「新しいAPIキーを発行」をクリック
4. 権限で「資産の参照」と「注文（新規・取消）」に必ずチェック（売買に必要）
5. 発行された API Key と Secret を上のフォームに貼り付けて保存

❓ このBotは何をする？

GMOとbitbankのETH価格差（乖離）を利用して、bitbankで買い注文と売り注文を並べ、その利ざやを狙う「マーケットメイキング」を自動で行います。相場を4段階で判定し、危険な時は自動で発注を止めます。

⚠️ 暗号資産の自動売買は損失のリスクがあります。本ツールは教育・研究目的であり、投資助言ではありません。実際の運用は自己責任で行ってください。必ずシミュレーションで確認してから本番に進んでください。
