# 🤖 crypto-mm-bot — bitbank マーケットメイキングBot

GMO と bitbank の価格差（乖離）を利用して、ETH/JPY で自動マーケットメイキング（指値の買い・売りを並べて利ざやを狙う）を行う自動売買ボットです。4段階の相場判定・シミュレーションモード・Web監視ダッシュボードつき。

---

## ⚠️ 最初に必ずお読みください

- 本ツールは**教育・研究目的**です。投資助言ではありません。実取引は**自己責任**で行ってください。
- 暗号資産の自動売買は**損失のリスク**があります。まず必ず**シミュレーションモード**で動作を確認してください。
- APIキーは**あなた自身の bitbank アカウント**のものを使います。第三者に渡さないでください。

---

## 🖥️ どこに設置するか

このボットは **24時間動かし続ける**必要があります（10秒ごとに価格を見て売買するため）。
そのため、**ご自身が契約したレンタルサーバー（VPS）に設置してください。**

- 例：さくらのVPS / ConoHa VPS / Xserver VPS など、Python が動くレンタルサーバー
- お試しだけなら、自宅PCでも動きます（PCの電源を切ると停止します）

> 販売元のサーバーは使いません。必ずご自身のレンタルサーバー・PCで動かしてください。

## 🚀 かんたん設置ガイド（4ステップ）

### 1. ファイルを置く
契約したレンタルサーバーに、このフォルダ一式をアップロードします。

### 2. APIキーを設定する
`.env.example` をコピーして `.env` を作り、あなたの bitbank APIキーを記入します。

```bash
cp .env.example .env
```

`.env` の中身：
```
BITBANK_API_KEY=あなたのAPIキー
BITBANK_API_SECRET=あなたのAPIシークレット
```

> bitbank の APIキーは、bitbank にログイン →「API」メニュー → 新規発行 で取得できます。
> **「注文」権限にチェック**を入れて発行してください（売買するため）。

### 3. 必要なものを入れる
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 4. 動かす

**まずはシミュレーション（お金を使わずテスト）：**
```bash
python3 demo.py
```

**本番（実際に売買）：**
```bash
python3 bot_v2.py
```

## 📊 監視ダッシュボード（web/）

`web/` フォルダを PHP が動くサーバーに置くと、ブラウザで稼働状況（価格・乖離・損益・約定）を確認できます。

- `web/index.html` … 監視画面
- `web/api.php` … 状態を返すAPI

ブラウザで `https://あなたのサーバー/web/` を開いてください。

## ⚙️ 主な設定（bot_v2.py 上部）

| 設定 | 意味 | 初期値 |
|---|---|---|
| `SYMBOL` | 通貨ペア | eth_jpy |
| `ORDER_SIZE` | 1回の注文量 | 0.05 ETH |
| `INTERVAL` | 価格チェック間隔（秒） | 10 |
| `SPREAD_RANGE` | レンジ相場のスプレッド | 300円 |
| `SPREAD_TREND` | トレンド相場のスプレッド | 800円 |
| `CRASH_THRESHOLD` | 急落判定（%） | 3% |

はじめは初期値のまま、シミュレーションで様子を見ることをおすすめします。

---

## 🧪 4段階の相場判定

1. **乖離チェック** — GMOとbitbankの価格差が十分あるか
2. **ボラティリティ** — 値動きが激しすぎないか
3. **レンジ/トレンド** — 相場の種類を判定してスプレッドを調整
4. **急落ガード** — 暴落時は自動で発注を止める
